# Home-furniture 
# 🪑 PHP Home Furniture Project

## 📖 Project Overview
The **Home Furniture Project** is a mini **E-Commerce Web Application** developed using **PHP and MySQL**.  
It is designed to manage and display furniture products (sofas, tables, chairs, etc.), allow customers to browse items, add them to a shopping cart, and complete checkout.  

The project also includes an **Admin Panel** where the admin can add, update, or remove furniture products and manage the website content.

This project is useful for:
- Learning **PHP + MySQL integration**
- Understanding **e-commerce workflows** (cart, checkout, admin panel)
- Practicing **session handling, authentication, and CRUD operations**

---

## ✨ Features (Detailed)

### 👤 User Side (Frontend)
1. **Home Page**
   - Attractive homepage showcasing categories like *Sofa, Tables, Chairs, Beds, etc.*
   - Navigation menu (Home, About, Gallery, Blog, Contact, etc.)

2. **Product Catalog**
   - Category-wise browsing of furniture.
   - Pages like `pview.php`, `pviewsofa.php` display details of each product.
   - Images, description, and price of each product.

3. **Shopping Cart**
   - Add multiple products to cart.
   - Increase/decrease quantity or remove items.
   - Total bill calculation.

4. **Checkout System**
   - User provides address, contact info, and payment details.
   - Order confirmation after placing the order.

5. **User Authentication**
   - Registration (`register.php`) and login system (`login.php`).
   - Secure session handling.
   - Logout feature.

6. **Extra Pages**
   - **Blog Page** – Furniture ideas and tips.
   - **Gallery Page** – Showcase of furniture items.
   - **Testimonials** – Customer feedback page.
   - **Contact Us** – Form for user inquiries.

---

### 🛠 Admin Panel (Backend)
1. **Admin Login (`admin.php`)**
   - Secured login for admin.
   - Only authenticated admins can access the dashboard.

2. **Product Management**
   - Add new furniture items with image, description, and price.
   - Edit/update existing products.
   - Delete furniture items.

3. **Order Management**
   - View orders placed by users.
   - Manage order status (Pending, Completed).

---

## 🛠 Tech Stack
- **PHP (7.x / 8.x)**  
- **HTML, CSS, JavaScript** (Frontend)  
- **MySQL Database**  
- **Apache / XAMPP / WAMP** (Server)  

---

## ⚙️ Prerequisites
- Install [XAMPP](https://www.apachefriends.org/) or WAMP
- Enable **Apache** and **MySQL**
- Import `furniture.sql` into phpMyAdmin
- Configure `config.php` with your DB credentials
